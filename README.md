| 功能模块 | 说明 |
|---------|------|
| 🎯 **智能匹配** | 输入分数自动估算位次，或直接输入位次反推分数（物理/历史双科支持） |

## 💻 本地使用

1. 下载 `河北高考志愿助手.html`
2. 用 Chrome / Edge 浏览器打开即可运行
3. **所有数据内嵌**，无需网络，离线可用
4. 手机端也可用，适配移动端布局

## 🔧 技术特色

- ✅ **纯前端**：HTML + CSS + JavaScript，零依赖
- ✅ **单文件**：所有数据内嵌，一个文件搞定
- ✅ **759条数据**：院校信息压缩后约180KB
- ✅ **分段线性插值**：分数到位次换算，46个校准点
- ✅ **四种排序算法**：灵活切换，满足不同策略需求
- ✅ **事件委托**：性能优化，千条数据流畅运行
- ✅ **响应式设计**：手机/平板/电脑均可使用

## 📜 数据来源

- 📊 河北省教育考试院：历年高考分数线
- 📊 麦可思《2025年中国本科生就业报告》：薪资数据
- 📊 各院校公开招生数据：录取分数与位次（估算值）
- 📊 百科/贴吧/招生简章：宿舍条件信息

> **⚠️ 声明**：数据为估算值，仅供参考。正式填报请以河北省教育考试院官方公布为准。
