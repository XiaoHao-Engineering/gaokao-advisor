## 为什么需要这个工具？
每年河北高考生拿到分数后最焦虑的就是：我的分数能上什么学校？这个工具帮你：
- 输入分数 自动换算位次 匹配院校
- 含宿舍条件查询（几人寝 上床下桌）
- 物理类/历史类双轨数据，文理考生都能用
- 含专科推荐，分数低至200分

## 快速开始
1. 选省份 + 选选科 + 选排序策略
2. 输入分数或位次
3. 点击开始匹配

[在线体验](https://xiaohao-engineering.github.io/gaokao-advisor/)

## 核心功能
|功能|说明|
|-|-|
|759所院校|本科654所+专科105所覆盖26省|
|张雪峰策略|系统根据分数段自动推荐冲学校还是保专业|
|宿舍查询|点击院校详情可查看几人寝上床下桌|
|文理双轨|物理类+历史类各自独立数据|
|排序策略|专业/学校/城市/智能推荐四种模式|
|专科推荐|105所双高示范性专科分数低至200分|
|就业参考|内置麦可思2025薪资数据|
